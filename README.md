# kotlin-library-reading-room

### 프로젝트 개요

- 코틀린 언어로 객체지향적인 도서관 열람실 좌석 배정 프로그램 구현하기

### 목표

- TDD 원칙에 따라 red-green-refactor 단계별로 커밋 남기기
- 메서드가 15라인을 넘지 않도록 작성하기
- 단일책임원칙을 준수하여 클래스 작성하기

### 구현할 기능 목록

사용자 프롬프트 기능
- [x] 메뉴 출력 및 입력
  - 사용자로부터 기능에 해당하는 번호를 입력 받는다.
  - [예외] 존재하지 않는 번호를 입력하면 에러 메시지를 출력하고 다시 입력 받는다.
  - [예외] 입력된 값이 번호가 아니면 예외가 발생한다
- [x] 열람실 좌석 현황 조회 기능
  - 열람실에 있는 좌석들을 출력하고 메뉴로 이동한다
- [x] 열람실 좌석 배정 기능
  - 배정된 좌석이 있으면 메시지를 출력하고 메뉴로 이동한다
  - 좌석 번호를 입력받으면 좌석을 배정 처리하고 메뉴로 이동한다.
    - [예외] 열람실에 존재하지 않는 번호를 입력하면 에러 메시지를 출력하고 메뉴로 이동한다.
    - [예외] 좌석번호가 1 미만이면 예외 메시지를 출력하고 메뉴로 이동한다
    - [예외] 좌석을 하나를 초과해서 배정받으려고 하면 예외 메시지를 출력하고 메뉴로 이동한다
- [x] 열람실 좌석 반납 기능
  - 반납할 좌석을 출력하고, 좌석을 반납할지 여부를 Y(예) 또는 n(아니오) 형태로 입력 받는다.
    - Y를 입력 받으면 좌석 반납 완료 메시지를 출력하고 메뉴로 이동한다.
    - n을 입력하면 메시지를 출력하고 메뉴로 이동한다.
    - [예외] 정해진 입력이 아니면 에러 메시지를 출력하고 다시 입력받는다.
  - 반납할 좌석이 없으면 반납할 좌석이 없다고 출력하고 메뉴로 이동한다
- [x] 사용자 메뉴 종료 기능
  - 사용자 메뉴를 종료한다

로그인 및 회원가입 프롬프트 기능
- [x] 메뉴 출력 및 입력
  - 사용자에게 기능 목록을 출력하고 기능에 해당하는 번호를 입력받는다.
    - [예외] 존재하지 않는 번호를 입력하면 에러 메시지를 출력하고 다시 입력 받는다.
    - [예외] 입력된 값이 번호가 아니면 예외가 발생한다
- [ ] 사용자 로그인 
  - 시스템이 기억하는 현재 시각을 출력한 후에 학번과 비밀번호, 현재 시간을 입력받는다
  - 시간은 YYYY-mm-dd hh:mm:ss 형태로 입력받는다
    - [예외] 시간을 형식이 올바르지 않으면 에러 메시지를 출력하고 메뉴로 이동한다.
    - [예외] 시스템이 기억하는 시간보다 과거의 시간이면 에러 메시지를 출력하고 메뉴로 이동한다.
  - 일치하는 계정이 있으면 로그인 상태로 처리하고 사용자 프롬프트를 출력한다
  - 일치하는 계정이 없으면 에러 메시지를 출력하고 메뉴로 이동한다.
- [x] 회원가입
  - 시스템이 기억하는 현재 시각을 출력한 후에 새로운 계정의 학번, 이름, 비밀번호, 그리고 현재시간을 입력받는다.
    - [예외] 학번이 일치하는 사용자가 있으면 에러 메시지를 출력하고 메뉴로 이동한다.
    - [예외] 학번이 9자리 숫자가 아니면 에러 메시지를 출력하고 메뉴로 이동한다
    - [예외] 이름은 길이가 1 이상이고 공백이 아니다. 이를 위배하면 에러 메시지를 출력하고 메뉴로 이동한다.
    - [예외] 비밀번호가 영어대소문자와 숫자로만 이루어진 8자리에서 12자리 문자열이 아니면 에러 메시지를 출력하고 메뉴로 이동한다.
  - 시간은 YYYY-mm-dd hh:mm:ss 형태로 입력받는다
    - [예외] 시간을 형식이 올바르지 않으면 에러 메시지를 출력하고 메뉴로 이동한다.
    - [예외] 시스템이 기억하는 시간보다 과거의 시간이면 에러 메시지를 출력하고 메뉴로 이동한다.
  - 회원가입에 성공하면 메뉴로 이동한다.
